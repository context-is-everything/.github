# Context is Everything

We build **Sasha Studio** — an AI knowledge management platform that unlocks decades of institutional expertise through conversational AI. Sasha connects to your existing documents, learns your organisation's history, and makes that collective intelligence instantly accessible to every team member.

### Get involved

- **Website**: [context-is-everything.com](https://www.context-is-everything.com)
- **Community Skills**: Browse and contribute agent skills in the [skills](https://github.com/context-is-everything/skills) repo

We welcome contributions — whether it's new agent skills, bug reports, or feature ideas. Open an issue or submit a PR to get started.
